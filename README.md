The **Juice Crimea Cola** project is aimed at developing a web application with a beautiful user interface. My goal is to replicate everything with enthusiasm and then delve into learning Redux.

## About the Project

- **Name:** Juice Crimea Cola
- **Objective:** Create a web application with an appealing UI
- **Motivation:** The project is done for fun and personal growth
- **Plan:**
  1. Replicate everything with enthusiasm

## Technologies

- **Frontend:**
  - React (with prior experience)
  - CSS (with prior experience)

## How to Get Involved

1. Clone the repository:
   ```
   git clone https://github.com/Mihuil121/juice-Crimea-cola.git
   ```

2. Install dependencies:
   ```
   cd juice-Crimea-cola
   npm install
   ```

3. Run the application:
   ```
   npm start
   ```
